# ✦ FlowSpace — Local

Tu segundo cerebro. 100% local, sin cuentas, sin internet.

## Instalación (un comando)

```bash
curl -fsSL https://raw.githubusercontent.com/mason99-stack/flowspace/main/install.sh | bash
```

## Instalación manual

```bash
git clone https://github.com/mason99-stack/flowspace
cd flowspace
npm install
npm run build
npm start
```

Abre http://localhost:3000

## Uso después de instalar

```bash
flowspace
```

## ¿Dónde se guardan los datos?

```
~/.flowspace/data.json
```

Puedes hacer backup exportando desde Ajustes → Exportar JSON.

## Requisitos

- Node.js 18+
- git
